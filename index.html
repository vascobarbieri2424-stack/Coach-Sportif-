Comment utiliser ce programme
🏋️
Mode Salle : chaque semaine a un bouton toggle. Clique "Mode Salle" pour voir uniquement les exercices avec équipement (machines, barres).
🌳
Mode Extérieur : chaque exercice a une alternative sans salle — poids de corps, sac lesté, barre de parc. Clique "Mode Extérieur".
📋
Tests S1 + S9 + S14 : remplir les tableaux au fur et à mesure. La comparaison sur 3 colonnes (S14) montrera tout le travail accompli.
🔄
Sans club après S14 : blocs rotatifs A (développement) → B (consolidation) → C (fraîcheur / essai direct). Semaine C = disponible pour un essai.
📁
Fonctionnement : tous les fichiers semaine*.html doivent être dans le même dossier que cet index.html pour que la navigation fonctionne.
⚡
Progression : la pyramide — Mouvement → Robustesse → Cardio → Force → Puissance. L'ordre ne peut pas être sauté. S11 est le premier travail réellement explosif.
